# 数学打表解
核心问题是判断一个数字 $n$ 是不是有且只有4个因数。
首先一个数字必定有1和自己两个因数
剩下的两个因数不能“引入”新的因数
所以当一个数字 $n$ 有且仅有4个因数时，它必定是以下几种情况
- 两个质数的乘积
- 一个质数的立方
- 两个质数的乘积和其中任意一个质数的乘积 -> 不成立，因为此时有$A, B, A^2$或者$B^2$ 3个额外因数

有质数$A$ $B$，能组合出来的只有 $A*B, A^3, B^3$ 3个情况
注意到$n$小于等于$10^5$，能不能基于这个规律直接打表？

$10^5$ 以下所有的质数有好多，感觉打表会OOM

# 数学方法
基于上述结论
我要检查一个 $n$，其实就是检查 $\sqrt{n}$ 向下取整的因数里是否有且仅有1个数，如果是，那么就是4因数。

然后因为n会重复，要记一下已经算过的值

# 更暴力的办法
大小只有 $10^5$ 直接打表，但是不打因数表，而是直接打结果表
维护
counter[i] = i有几个因数
sum[i] = i的因数的合
for i in range(1, 100001):
    for j in range(i, 100001, i):
        counter[j] += 1  # 走i步必然是有i这个因数
        sum[j] += i

然后直接查询counter[n] == 4即可
