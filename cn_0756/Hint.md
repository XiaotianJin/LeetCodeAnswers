# 暴力法
- 这题的暴力法肯定是超时的
- 用一个倒置的矩阵 $n*n$ 来模拟金字塔的情况
- 第 $i$ 行第 $j$ 个元素是通过 第 $i-1$ 行第 $j$ 和第 $j+1$ 个元素去 $allowed$ 来获取的
- 这样的暴力法时间复杂度是 $O(n*(n*m)), m= len(allowed)$

# 优化
- $allowed$ 里给出的是组合，$ABC$ 其实可以认为是 $A+B = C$这样的算式，注意AB不能互换
- 我们要做的事找到可以用这些算式拼接出的金字塔
- 每一行可能的所有结果还是得暴力出来，如果某一行的所有结果里都没有满足算式左侧的组合，就不能拼出金字塔
- 找到所有可能的组合，再进行检查，直到只剩下最后一行
